Offer Catalog by Event

Salary Increment

  • Primary: SmartInvest (robo/portfolios), ILP/term top‑up, HYSA auto‑sweep
  • Incentives: fee waiver 3m, step‑up APY, “invest your raise” badge
  • Guardrails: DTI < 35%, credit band min

Account Exhaust

  • Primary: micro LOC, overdraft with rails, expense coach
  • Incentives: 0% intro 30d, helpful tips over hard sell
  • Guardrails: hardship flags → suppress sales

Travel Spend

  • Primary: FX wallet + multi‑currency card, travel insurance, premium travel card
  • Incentives: lounge passes, FX fee rebate, partner miles
  • Guardrails: sanctions/geo checks; CN mini‑program variants

High‑Value Txn

  • Primary: BNPL/EMI conversion, lifestyle card upgrade, device protection
  • Incentives: 1‑click EMI, tenure personalization
  • Guardrails: affordability & KYC re‑check; merchant risk

High Purchase Frequency

  • Primary: premium cashback card, curated partner bundles
  • Incentives: stacked cashback week, creator collabs
  • Guardrails: fatigue cap < 3 impressions / 7d

Data & Feature Store

  • Streams: cards/UPI, ledger, payroll, device/app analytics, RM notes, market/FX, partners.
  • Key features: spend_share_by_MCC_30d, volatility_ratio, salary_growth_Δ%, days_of_cash_left, product_holdings_onehot, response_history, sentiment, risk band.
  • Storage: offline parquet + online feature store (Feast); TTL; p50 read < 30ms.

Models & CSI

  • Propensity (30/60/90d): XGBoost/LightGBM; calibrated (Platt/Isotonic).
  • Embeddings: DeepFM/Autoencoder; HDBSCAN micro‑segments.
  • NLP: EventNER & sentiment with risk/toxicity filters.
  • CSI (0–100): meta‑ensemble over behavior/affinity/life‑events/engagement/sentiment/top‑propensity; weekly re‑learn.
  • Latency budget (p95): detect ≤150ms → features ≤60ms → models ≤120ms → RTIM ≤150ms → AEM ≤200ms → render ≤200ms.

RTIM Policy & Frequency Caps

  • Eligibility: product_eligibility ∩ regulatory_ok ∩ risk_ok ∩ contact_policy_ok
  • Tie‑breakers: expected_uplift×margin – cost → novelty → channel readiness
  • Cadence: ≤1 high‑impact offer/day; 3 impressions/week; ≤1 push within 24h
  • Suppression: 22:00–08:00 quiet hours; complaint/decline flags
  • Exploration: Thompson Sampling 10–20% for variant learning

Mobile Journeys

Salary Increment (High CSI)

  1. Celebratory tile → raise projection (“Invest extra $X → +$Y in 10y”).
  2. 1‑tap goal (house/travel/skills) → prefilled SIP.
  3. “Why this offer?” (XAI): salary Δ%, surplus trend, disclosure.
  4. Invisible soft checks → instant onboarding; fallback to education path.

Account Exhaust (Med CSI)

“Safety net” coach → overdraft sliders + repayment forecast; “bill smoothing” alternative.

Travel Spend

Trip card → prefilled wallet currencies, airline perks; 1‑tap insurance toggle.

High‑Value Txn

Post‑purchase snackbar → “Split this payment?” with personalized tenors.

Governance, Risk & Compliance

  • Explicit consent & purpose; per‑market data residency.
  • PII minimization; tokenized IDs; strict RBAC.
  • Fairness: parity checks; mitigate if disparate impact > 5%.
  • XAI: in‑app explanation strings from SHAP summaries.
  • Audit: immutable logs (RTIM decisions, model versions, features).
  • Edge cases: sanctioned geos, hardship flags → suppress sales offers.

Measurement & Experimentation

  • Primary: conversion, latency, engagement, AUC; Secondary: opt‑in, fatigue, CAC payback.
  • Design: CUPED / geo‑split; power for 1.5–2.0 pp uplift @ 90% power; sequential tests.
  • Dashboards: event counts, offer funnel, CSI drift, calibration/SHAP drift, suppress reasons.

Tech Blueprint

  • Streaming: Kafka + Flink
  • Feature Store: Feast + Redis (online)
  • Models: Python/MLflow, XGBoost/LightGBM, ONNX/Triton serving
  • RTIM: Adobe Journey Optimizer / Pega NBA
  • AEM: Headless Content Fragments (persona × event × tier × locale)
  • App: React Native/native; server‑driven UI tiles; deep links
  • Observability: OpenTelemetry, Prometheus, Grafana
